Sobre o Projeto

Este trabalho tem como objetivo demonstrar as habilidades adquiridas no desenvolvimento de páginas web. Através deste projeto, será possível aplicar os conceitos de estruturação de conteúdo com HTML, estilização e design com CSS, e interatividade com JavaScript. A proposta é criar uma página funcional e visualmente atraente, explorando as melhores práticas de desenvolvimento web e a integração entre essas três tecnologias, de forma a apresentar um site que seja tanto bem estruturado quanto dinâmico.
